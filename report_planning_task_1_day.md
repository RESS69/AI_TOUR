## แพ็คเกจท่องเที่ยว 1 วัน ภาคตะวันออก (5 จังหวัด)


**แพ็คเกจที่ 1:  เสน่ห์แห่งทะเลระยอง**

* **เวลา:** 07:00 - 18:00 น.
* **สถานที่:** จังหวัดระยอง
* **กิจกรรม:**
    * 07:00 - 08:00 น. :  เดินทางจากจุดนัดพบไปยังหาดแม่รำพึง
    * 08:00 - 10:00 น. :  เล่นน้ำทะเล อาบแดด พักผ่อนบนหาดแม่รำพึง
    * 10:00 - 11:00 น. :  รับประทานอาหารเช้า ณ ร้านอาหารริมทะเล
    * 11:00 - 13:00 น. :  เที่ยวชมและถ่ายรูปที่แหลมแม่พิมพ์ ชมความสวยงามของหาดทรายและธรรมชาติ
    * 13:00 - 14:00 น. :  รับประทานอาหารกลางวัน ณ ร้านอาหารทะเลสดๆ
    * 14:00 - 16:00 น. :  เยี่ยมชมสวนพฤกษศาสตร์ระยอง
    * 16:00 - 17:00 น. :  แวะซื้อของฝาก ณ ตลาดน้ำระยอง
    * 17:00 - 18:00 น. :  เดินทางกลับ


**แพ็คเกจที่ 2:  สัมผัสวัฒนธรรมจันทบุรี**

* **เวลา:** 07:00 - 18:00 น.
* **สถานที่:** จังหวัดจันทบุรี
* **กิจกรรม:**
    * 07:00 - 08:00 น. :  เดินทางจากจุดนัดพบไปยังวัดพระศรีมหาธาตุวรวิหาร
    * 08:00 - 09:30 น. :  เที่ยวชมวัดพระศรีมหาธาตุวรวิหาร ชมความงดงามของสถาปัตยกรรม
    * 09:30 - 10:30 น. :  เยี่ยมชมพิพิธภัณฑสถานแห่งชาติจันทบุรี
    * 10:30 - 11:30 น. :  รับประทานอาหารเช้า ณ ร้านอาหารท้องถิ่น
    * 11:30 - 13:30 น. :  เที่ยวชมเมืองจันทบุรี ชมบ้านเรือนโบราณและวิถีชีวิตชาวบ้าน
    * 13:30 - 14:30 น. :  รับประทานอาหารกลางวัน ณ ร้านอาหารขึ้นชื่อ
    * 14:30 - 16:30 น. :  เที่ยวชมสวนรุกขชาติเขาสอยดาว
    * 16:30 - 17:30 น. :  แวะซื้อของฝาก ณ ตลาดท่าใหม่
    * 17:30 - 18:00 น. :  เดินทางกลับ


**แพ็คเกจที่ 3:  ธรรมชาติและประวัติศาสตร์สระแก้ว**

* **เวลา:** 07:00 - 18:00 น.
* **สถานที่:** จังหวัดสระแก้ว
* **กิจกรรม:**
    * 07:00 - 08:30 น. :  เดินทางจากจุดนัดพบไปยังปราสาทสด๊กก๊อกธม
    * 08:30 - 10:30 น. :  เที่ยวชมปราสาทสด๊กก๊อกธม  เรียนรู้ประวัติศาสตร์
    * 10:30 - 11:30 น. :  รับประทานอาหารเช้า ณ ร้านอาหารท้องถิ่น
    * 11:30 - 13:30 น. :  เยี่ยมชมเขื่อนเขาฉกรรจ์
    * 13:30 - 14:30 น. :  รับประทานอาหารกลางวัน ณ ร้านอาหารริมเขื่อน
    * 14:30 - 16:30 น. :  เที่ยวชมอุทยานแห่งชาติตาพระยา
    * 16:30 - 17:30 น. :  แวะซื้อของฝาก ณ ตลาดชายแดน
    * 17:30 - 18:00 น. :  เดินทางกลับ


**แพ็คเกจที่ 4:  เสน่ห์เมืองชลบุรี**

* **เวลา:** 07:00 - 18:00 น.
* **สถานที่:** จังหวัดชลบุรี
* **กิจกรรม:**
    * 07:00 - 08:00 น. :  เดินทางจากจุดนัดพบไปยังวัดญาณสังวรารามวรมหาวิหาร
    * 08:00 - 09:30 น. :  เที่ยวชมวัดญาณสังวรารามวรมหาวิหาร
    * 09:30 - 10:30 น. :  เยี่ยมชมตลาดน้ำ 4 ภาค
    * 10:30 - 11:30 น. :  รับประทานอาหารเช้า ณ ร้านอาหารในตลาด
    * 11:30 - 13:30 น. :  เที่ยวชมหาดบางแสน
    * 13:30 - 14:30 น. :  รับประทานอาหารกลางวัน ณ ร้านอาหารริมทะเล
    * 14:30 - 16:30 น. :  เยี่ยมชมพิพิธภัณฑ์ชลบุรี
    * 16:30 - 17:30 น. :  แวะซื้อของฝาก ณ ตลาดนัดบางแสน
    * 17:30 - 18:00 น. :  เดินทางกลับ


**แพ็คเกจที่ 5:  ความงดงามของฉะเชิงเทรา**

* **เวลา:** 07:00 - 18:00 น.
* **สถานที่:** จังหวัดฉะเชิงเทรา
* **กิจกรรม:**
    * 07:00 - 08:00 น. :  เดินทางจากจุดนัดพบไปยังวัดสมานรัตนาราม
    * 08:00 - 09:30 น. :  เที่ยวชมวัดสมานรัตนาราม
    * 09:30 - 10:30 น. :  เยี่ยมชมตลาดน้ำบางน้ำเปรี้ยว
    * 10:30 - 11:30 น. :  รับประทานอาหารเช้า ณ ร้านอาหารในตลาด
    * 11:30 - 13:30 น. :  เที่ยวชมเมืองโบราณ
    * 13:30 - 14:30 น. :  รับประทานอาหารกลางวัน ณ ร้านอาหารในตัวเมือง
    * 14:30 - 16:30 น. :  เยี่ยมชมอุทยานแห่งชาติเขาใหญ่ (ส่วนที่อยู่ในฉะเชิงเทรา)
    * 16:30 - 17:30 น. :  แวะซื้อของฝาก ณ ร้านค้าท้องถิ่น
    * 17:30 - 18:00 น. :  เดินทางกลับ


**หมายเหตุ:**  เวลาอาจมีการปรับเปลี่ยนเล็กน้อย ขึ้นอยู่กับสภาพการจราจรและความพร้อมของสถานที่  ควรตรวจสอบข้อมูลการเปิด-ปิด ของสถานที่ท่องเที่ยวล่วงหน้า  ราคาทัวร์และค่าใช้จ่ายอื่นๆ  ขึ้นอยู่กับจำนวนผู้ร่วมทริปและความต้องการของลูกค้า  ควรติดต่อผู้ให้บริการทัวร์เพื่อขอรายละเอียดเพิ่มเติม
